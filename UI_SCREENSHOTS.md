## Features:

✅ **Real-time Updates** - Auto-refresh ทุก 5 วินาที
✅ **Responsive Design** - ใช้งานได้ทั้ง Desktop, Tablet, Mobile
✅ **System Status** - แสดงสถานะ ONLINE/OFFLINE
✅ **Camera Cards** - แสดงรายละเอียดแต่ละกล้อง
✅ **Color-coded Events** - แยกสีตาม event type
✅ **Clean UI** - ดูง่าย ไม่ซับซ้อน

✅ **Advanced Charts** - Line graphs, histograms, heatmaps

✅ **Variables** - Filter by camera, date range

✅ **Sharing** - Share dashboard link
